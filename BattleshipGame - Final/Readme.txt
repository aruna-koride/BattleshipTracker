Instructions:
1.This progam is created using VisualStudio 2019.
2.It's a console application.
3.Run the program and follow the instructions on the console.

Assumptions: 
1.Creating Board and Add ships are done automatically in the code.
2.Each row of the boardgame is assigned a row header[A-J] and columns are in range[0-9].
3.The target square to hit is chosen by passing the respective row header and column index values(Ex:F,6).
4.Receiving attack is only tested at Row-F with the column indexes randomly set. 
5.The player is assumed to be lost when all the ships are sunk.
